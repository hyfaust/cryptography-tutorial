## Introduction

This tutorial takes you from zero knowledge to a deep understanding of modern cryptography through **25 progressively challenging topics** organized into 7 modules. Each topic includes:

- Detailed explanations of core concepts and terminology
- Hands-on experiments using industry-standard tools
- Executable Python demonstration scripts
- Practice exercises and further reading

## Learning Roadmap

## Modules

| Module | Topics | Key Concepts |
|--------|--------|-------------|
| **1 — Foundations** | 5 | Classical ciphers, Caesar, Vigenère, encoding vs encryption |
| **2 — Hashing** | 4 | MD5, SHA-256, collisions, HMAC, password hashing, salt |
| **2b — CSPRNG** | 3 | TRNG/PRNG/CSPRNG, HMAC-DRBG, randomness attacks |
| **3 — Symmetric Encryption** | 4 | DES, AES, block cipher modes (ECB/CBC/CTR/GCM), stream ciphers |
| **4 — Asymmetric Encryption** | 4 | Number theory, RSA, ECC, Diffie-Hellman |
| **5 — Cryptanalysis** | 5 | Frequency analysis, hashcat, John the Ripper, RSA attacks |
| **6 — Advanced** | 5 | Digital signatures, PKI, TLS/SSL, zero-knowledge proofs, post-quantum |

## Installation

```bash
# Clone or download the project
cd cryptography_learn/crypto-tutorial

# Install Python dependencies
pip install cryptography pycryptodome sympy

# Build the site
mkdocs build

# Serve locally
mkdocs serve -a 127.0.0.1:8888
```

Open your browser and navigate to `http://127.0.0.1:8888`.

## Usage

### Reading the Documentation

The built site provides:

- **Dark/light theme toggle** in the header
- **Code copy buttons** on all code blocks
- **Mermaid diagrams** for visual explanations
- **KaTeX math formulas** for mathematical notation
- **Tabbed interfaces** for multi-tool demonstrations
- **Full-text search** across all pages

### Running the Scripts

All 26 Python scripts are in the `docs/scripts/` directory:

```bash
cd docs/scripts

# Classical cipher demos
python caesar_cipher.py --encrypt "HELLO" --key 3
python vigenere_cipher.py --encrypt "HELLO" --key "SECRET"

# Hash demos
python hash_demo.py
python birthday_attack.py --attack

# CSPRNG demos
python random_demo.py --predict
python randomness_attack.py --ecdsa

# Symmetric encryption demos
python aes_demo.py
python block_modes_demo.py

# Number theory & asymmetric demos
python number_theory_demo.py
python rsa_demo.py
python ecc_demo.py
python dh_demo.py

# RSA attack demos
python rsa_small_e.py
python rsa_common_modulus.py
python rsa_wiener.py

# Advanced demos
python rsa_signature.py
python ecdsa_demo.py
python zkp_demo.py
```
