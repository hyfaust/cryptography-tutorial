    AESзҡ„еҜҶй’Ҙи°ғеәҰ¶

    AESзҡ„еҜҶй’Ҙи°ғеәҰз®—жі•з”ҹжҲҗиҪ®еҜҶй’Ҙпјҡ

    AES-128еҜҶй’Ҙи°ғеәҰ¶

    • иҫ“е…Ҙпјҡ128дҪҚеҜҶй’ҘпјҲ4дёӘ32дҪҚеӯ—пјү
    • иҫ“еҮәпјҡ11дёӘиҪ®еҜҶй’ҘпјҲжҜҸиҪ®128дҪҚпјү
    • жҜҸиҪ®з”ҹжҲҗ4дёӘ32дҪҚеӯ—

    еҜҶй’Ҙжү©еұ•иҝҮзЁӢпјҡ 1. еүҚ4дёӘеӯ—зӣҙжҺҘжқҘиҮӘеҜҶй’Ҙ 2. еҗҺз»ӯеӯ—йҖҡиҝҮйҖ’еҪ’и®Ўз®— 3. жҜҸ4дёӘеӯ—иҝӣиЎҢдёҖж¬ЎRconпјҲиҪ®еёёж•°пјүж“ҚдҪң

    AES-192е’ҢAES-256¶

    еҸҳз§Қ еҜҶй’Ҙй•ҝеәҰ еқ—еӨ§е°Ҹ иҪ®ж•° иҪ®еҜҶй’Ҙж•°
    AES-128 128дҪҚ 128дҪҚ 10иҪ® 11дёӘ
    AES-192 192дҪҚ 128дҪҚ 12иҪ® 13дёӘ
    AES-256 256дҪҚ 128дҪҚ 14иҪ® 15дёӘ

    AESзҡ„ж•°еӯҰеҹәзЎҖ¶

    жңүйҷҗеҹҹ\(GF(2^8)\)¶

    AESдҪҝз”Ёжңүйҷҗеҹҹ\(GF(2^8)\)иҝӣиЎҢеӯ—иҠӮиҝҗз®—пјҡ

    • дёҚеҸҜзәҰеӨҡйЎ№ејҸпјҡ\(m(x) = x^8 + x^4 + x^3 + x + 1\)
    • еӯ—иҠӮиЎЁзӨәдёәеӨҡйЎ№ејҸзі»ж•°
    • д№ҳжі•дёәеӨҡйЎ№ејҸд№ҳжі•жЁЎ\(m(x)\)

    зӨәдҫӢпјҡ $$ {57} \cdot {83} = {C1} $$

    и®Ўз®—иҝҮзЁӢпјҡ 1. \(\{57\} = x^6 + x^4 + x^2 + x + 1\) 2. \(\{83\} = x^7 + x + 1\) 3. д№ҳз§ҜжЁЎ\(m(x)\)еҫ—еҲ°\(\{C1\}\)

    AESзҡ„е®Ңж•ҙеҠ еҜҶиҝҮзЁӢ¶

    graph TD
    A[128дҪҚжҳҺж–Ү] --> B[еҲқе§ӢиҪ®еҜҶй’ҘеҠ ]
    B --> C[з¬¬1иҪ®пјҡSubBytes]
    C --> D[ShiftRows]
    D --> E[MixColumns]
    E --> F[AddRoundKey]
    F --> G[...]
    G --> H[з¬¬9иҪ®пјҡSubBytes]
    H --> I[ShiftRows]
    I --> J[AddRoundKey]
    J --> K[128дҪҚеҜҶж–Ү]

    subgraph "дёӯй—ҙиҪ®пјҲз¬¬1-9иҪ®пјү"
        C --> D --> E --> F
    end

    subgraph "жңҖеҗҺдёҖиҪ®пјҲз¬¬10иҪ®пјү"
        H --> I --> J
    end

    еҠ еҜҶиҝҮзЁӢпјҡ 1. еҲқе§ӢиҪ®еҜҶй’ҘеҠ пјҡ\(State = Plaintext \oplus K_0\) 2. дёӯй—ҙиҪ®пјҲз¬¬1-9иҪ®пјүпјҡSubBytes вҶ’ ShiftRows вҶ’ MixColumns вҶ’ AddRoundKey 3. жңҖеҗҺдёҖиҪ®пјҲз¬¬10иҪ®пјүпјҡSubBytes вҶ’ ShiftRows вҶ’ AddRoundKeyпјҲж— MixColumnsпјү

    AESзҡ„и§ЈеҜҶиҝҮзЁӢ¶

    AESи§ЈеҜҶжҳҜеҠ еҜҶзҡ„йҖҶиҝҮзЁӢпјҡ

    1. йҖҶиҪ®еҜҶй’ҘеҠ 
    2. йҖҶShiftRows
    3. йҖҶSubBytes
    4. йҖҶMixColumnsпјҲеүҚ9иҪ®пјү

    и§ЈеҜҶйЎәеәҸпјҡ $$ Decrypt: AddRoundKey^{-1} \rightarrow MixColumns^{-1} \rightarrow ShiftRows^{-1} \rightarrow SubBytes^{-1} $$

    еңЁ OpenSSL 3.x дёӯпјҢopenssl enc е‘Ҫд»ӨдёҚеҶҚж”ҜжҢҒ AEAD жЁЎејҸпјҲеҰӮ GCMпјүгҖӮиҰҒдҪҝз”Ё GCM жЁЎејҸпјҢйңҖиҰҒдҪҝз”Ё OpenSSL зҡ„ EVP API жҲ–е…¶д»–е·Ҙе…·пјҲеҰӮ Python зҡ„ cryptography еә“пјүгҖӮ

    д»ҘдёӢе‘Ҫд»Өд»…йҖӮз”ЁдәҺ OpenSSL 1.x зүҲжң¬пјҡ

    # AES-GCMжҸҗдҫӣи®ӨиҜҒеҠ еҜҶ - д»…йҖӮз”ЁдәҺ OpenSSL 1.x

# и§ЈеҜҶе№¶йӘҢиҜҒ - д»…йҖӮз”ЁдәҺ OpenSSL 1.x

    AESзҡ„е®үе…Ёзү№жҖ§¶

    1. жҠ—е·®еҲҶеҜҶз ҒеҲҶжһҗ¶

    AESзҡ„и®ҫи®ЎиғҪеӨҹжҠөжҠ—е·®еҲҶеҜҶз ҒеҲҶжһҗпјҡ - S-Boxзҡ„жңҖеӨ§е·®еҲҶжҰӮзҺҮдёә\(2^{-6}\) - 4иҪ®AESзҡ„е·®еҲҶзү№еҫҒжҰӮзҺҮдёҚи¶…иҝҮ\(2^{-150}\)

    2. жҠ—зәҝжҖ§еҜҶз ҒеҲҶжһҗ¶

    AESзҡ„зәҝжҖ§иҝ‘дјјжҰӮзҺҮеҫҲдҪҺпјҡ - S-Boxзҡ„жңҖеӨ§зәҝжҖ§еҒҸе·®дёә\(2^{-3}\) - 4иҪ®AESзҡ„зәҝжҖ§зү№еҫҒжҰӮзҺҮдёҚи¶…иҝҮ\(2^{-75}\)

    3. жҠ—ж—Ғи·Ҝж”»еҮ»¶

    зҺ°д»ЈAESе®һзҺ°дҪҝз”Ёеёёж•°ж—¶й—ҙз®—жі•е’ҢжҺ©з ҒжҠҖжңҜжҠөжҠ—пјҡ - ж—¶й—ҙж”»еҮ» - зј“еӯҳж”»еҮ» - еҠҹиҖ—еҲҶжһҗ

    AESзҡ„ж”»еҮ»еҺҶеҸІ¶

    е°Ҫз®ЎAESи®ҫи®ЎзІҫиүҜпјҢд»ҚжңүдёҖдәӣзҗҶи®әж”»еҮ»пјҡ

    1. зӣёе…іеҜҶй’Ҙж”»еҮ»пјҲ2009пјүпјҡеҜ№AES-192е’ҢAES-256зҡ„зҗҶи®әж”»еҮ»
    2. ж—Ғи·Ҝж”»еҮ»пјҡй’ҲеҜ№зү№е®ҡе®һзҺ°зҡ„ж”»еҮ»
    3. йҮҸеӯҗи®Ўз®—еЁҒиғҒпјҡGroverз®—жі•е°ҶAES-128зҡ„е®үе…ЁжҖ§йҷҚиҮі64дҪҚ

    е®һйҷ…еҪұе“Қпјҡиҝҷдәӣж”»еҮ»еӨ§еӨҡжҳҜзҗҶи®әжҖ§зҡ„пјҢе®һйҷ…еә”з”ЁдёӯAESд»Қз„¶е®үе…ЁгҖӮ

    AESзҡ„еә”з”ЁеңәжҷҜ¶

    1. ж•°жҚ®еҠ еҜҶ¶

    • зЈҒзӣҳеҠ еҜҶпјҲBitLockerгҖҒFileVaultпјү
    • ж–Үд»¶еҠ еҜҶ
    • ж•°жҚ®еә“еҠ еҜҶ

    2. зҪ‘з»ңе®үе…Ё¶

    • TLS/SSLеҚҸи®®
    • VPNеҠ еҜҶ
    • Wi-Fiе®үе…ЁпјҲWPA2/WPA3пјү

    3. еә”з”ЁеұӮеҠ еҜҶ¶

    • ж¶ҲжҒҜеҠ еҜҶпјҲSignalгҖҒWhatsAppпјү
    • еҜҶз Ғз®ЎзҗҶеҷЁ
    • дә‘еӯҳеӮЁеҠ еҜҶ

    еҜҶй’Ҙй•ҝеәҰйҖүжӢ©жҢҮеҚ—¶

    е®үе…Ёзә§еҲ« жҺЁиҚҗеҜҶй’Ҙ иҜҙжҳҺ
    дёҖиҲ¬з”ЁйҖ” AES-128 и¶іеӨҹе®үе…ЁпјҢжҖ§иғҪжңҖдҪі
